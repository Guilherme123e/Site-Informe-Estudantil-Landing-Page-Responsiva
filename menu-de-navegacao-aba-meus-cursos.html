PYTHON

O curso de Python tem como objetivo ensinar os fundamentos da linguagem de programação Python, desde os conceitos básicos até tópicos mais avançados. Ao longo do curso, os alunos aprendem sobre variáveis, tipos de dados, estruturas de controle (condicionais e loops), funções, manipulação de arquivos, tratamento de erros, e bibliotecas padrão.

RUBY

O curso de Ruby busca introduzir a linguagem de forma prática e objetiva, abordando desde os primeiros passos até recursos mais sofisticados. Os participantes aprendem sobre sintaxe limpa, tipos de dados, estruturas de repetição e decisão, métodos, manipulação de arquivos, tratamento de exceções e uso de gems, que ampliam as funcionalidades da linguagem.

JAVA

O curso de Java tem como foco apresentar os principais conceitos da linguagem Java, desde a introdução até técnicas mais avançadas. Durante o curso, os alunos exploram a sintaxe da linguagem, tipos primitivos, estruturas condicionais e de repetição, criação de métodos, manipulação de arquivos, tratamento de exceções e o uso das bibliotecas padrão.

C++

O curso de C++ tem como proposta ensinar desde os fundamentos até aspectos mais técnicos da linguagem.
